Scalable E-Commerce Platform Deployment

This project involved designing, deploying, and operating a fully functional Kubernetes-based cloud infrastructure from scratch. The goal was to build a production-style container orchestration environment capable of running a scalable e-commerce application with persistent storage, load balancing, and external access.

The environment was deployed on CSC cPouta cloud using Infrastructure as Code. The final result was a working EverShop e-commerce platform backed by a PostgreSQL database, exposed via an Nginx load balancer and Kubernetes Ingress.

üèó Architecture Summary

The system was built using a multi-node Kubernetes cluster with clear separation of concerns:

  • Rancher Server ‚Äì Centralized Kubernetes cluster management
  • Kubernetes Master Node ‚Äì Control plane, etcd, and scheduling
  • Kubernetes Worker Node ‚Äì Application workloads
  • Nginx Load Balancer ‚Äì Public entry point for HTTP traffic
  • NFS Server ‚Äì Persistent shared storage for databases and application data

  • Kubernetes & Rancher ‚Äì Orchestration and lifecycle management
